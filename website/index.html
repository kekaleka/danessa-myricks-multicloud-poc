Danessa Myricks Multi-Cloud Proof of Concept

Demonstrating a multi-cloud architecture using AWS, Microsoft Azure, and Google Cloud Platform.

Project Overview

This project demonstrates how cloud resources from multiple cloud providers can be combined into a single cloud solution.

Cloud Providers

  • Amazon Web Services (AWS)
  • Microsoft Azure
  • Google Cloud Platform

Technologies Used

  • EC2
  • S3
  • Azure Database
  • Google Compute Engine
  • Cloud Storage
  • Ubuntu Linux
  • Nginx
