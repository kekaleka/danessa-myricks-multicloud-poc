# Danessa Myricks Multi-Cloud E-Commerce Proof of Concept

This project is a multi-cloud proof of concept designed for a beauty and cosmetics e-commerce brand. The goal was to demonstrate how AWS, Google Cloud Platform, and Microsoft Azure can work together to support web hosting, cloud storage, infrastructure, and database needs.

## Cloud Providers Used

- Amazon Web Services (AWS)
- Google Cloud Platform (GCP)
- Microsoft Azure

## Services Used

- AWS EC2
- AWS S3
- Google Compute Engine
- Google Cloud Storage
- Azure Database
- Ubuntu Linux
- Nginx

## Project Architecture

AWS and Google Cloud were used for web hosting and cloud storage, while Microsoft Azure was used for database support. This design demonstrated how a company could reduce dependency on one cloud provider and improve flexibility through a multi-cloud strategy.

## Skills Demonstrated

- Multi-cloud deployment
- Virtual machine configuration
- Linux server administration
- Nginx web hosting
- Cloud storage configuration
- Database planning
- Security groups and firewall rules
- Infrastructure documentation

## Project Purpose

This project shows how cloud infrastructure can support a scalable e-commerce environment for a cosmetics brand. It highlights cloud deployment, redundancy, and secure web hosting across multiple cloud providers.
