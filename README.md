# Serverless Expense Tracker

## Overview
A serverless expense tracking application built on AWS that allows users to create and retrieve expense records using AWS Lambda, API Gateway, and DynamoDB.

## Technologies
- AWS Lambda
- Amazon API Gateway
- Amazon DynamoDB
- HTML
- CSS

## Features
- Add new expenses
- Retrieve saved expenses
- Serverless architecture
- REST API
- Cloud-based data storage

## Project Structure

```
lambda/
    createExpense
    getExpenses

website/
    index.html
    style.css

docs/
    Expense tracker assignment.docx
```
